GET YOUTUBE SUBSCRIBERS

The API's provided by this application are following:

  • Get all subscribers: GET /subscribers
    Returns an array of all subscribers.
  • Get subscribers' names and subscribed channels: GET /subscribers/names
    Returns an array of subscribers with only their names and subscribed channels.
  • Get a subscriber by ID: GET /subscribers/:id
    Returns the subscriber with the given ID. If the ID does not match, returns a 400 status code with an error message.
